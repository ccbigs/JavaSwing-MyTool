## 1. 文字刷选功能

使用说明：
       将对应的关键字填入刷选框中，点击相应的软键，即可得出你所需求的关键字
  1、精确包含刷选:刷选出目标关键字中所包含相连的关键字
  2、不精确包含刷选:只要不包含相连的关键字就会被刷选出来
  3、包含刷选：刷选出目标关键字中含有的关键字
  4、非包含刷选：不包含一个字的关键字都会被刷选
